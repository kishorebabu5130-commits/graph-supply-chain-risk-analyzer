# Project Report

## Project Title

Graph-Based Supply Chain Risk Analyzer

## Problem Statement

Organizations face supply chain disruptions due to supplier failures, delays, and dependency risks. This project helps identify and predict risks using Graph Analytics and Machine Learning.

## Objectives

- Manage Suppliers
- Manage Dependencies
- Analyze Supply Chain Networks
- Predict Supplier Risk
- Generate Executive Reports

## Architecture

Frontend (Streamlit)

↓

FastAPI Backend

↓

SQLite Database

↓

NetworkX Graph Analytics

↓

Machine Learning Model

## Modules

### Supplier Management

CRUD operations for suppliers.

### Dependency Management

Create and manage supplier relationships.

### Graph Analytics

- Degree Centrality
- Betweenness Centrality
- PageRank

### Machine Learning

Random Forest model used for risk prediction.

### Dashboard

Executive insights and KPI metrics.

### Reporting

CSV exports and risk reports.

## Results

- 24 APIs implemented
- Dashboard implemented
- Machine Learning model implemented
- Graph analytics implemented

## Conclusion

The project successfully analyzes supply chain risks using graph-based analytics and machine learning techniques.
